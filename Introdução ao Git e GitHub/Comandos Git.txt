# Comandos para Git

- Git Status
- Git Add
- Git Commit = 
- Git Push
- Git Pull
- Git Clone
- mv = mover arquivo
- cd = mudança de diretório
- cd .. = voltar ao diretório anterior

- git add * = Adiciona todas as novas pastas

- git push origin main = empurra os arquivos para a nuvem (pasta principal (main))
